Build roadmap — click a phase

1

Core voice loop — start here

Wake word → STT → Claude → TTS

This is everything. One Python file that listens for "Hey Friday", turns your voice into text using Whisper, sends it to the Claude API, and speaks the reply back using pyttsx3.

Install:
pip install pvporcupine pip install openai-whisper pip install anthropic pip install pyttsx3 pip install sounddevice

Files you'll write:
main.py — the loop
stt.py — mic recording + Whisper
brain.py — Claude API call
tts.py — speaks the reply
2

PC automation

Open apps, create files, run code — no keyboard

Add a tool dispatcher in brain.py — Claude decides which tool to call based on your command. Then write individual tool functions.

Install:
pip install pyautogui pip install pygetwindow

Tools to write:
tools/open_app.py
tools/create_file.py
tools/run_code.py
tools/screen.py
3

Live data + browser automation

News, weather, Amazon orders — fully automated

Hook up live APIs for real-time info. Then add Playwright so Friday can operate websites like a human — search Amazon, place an order, reorder Swiggy, anything.

Install:
pip install playwright playwright install chromium pip install requests

Tools to write:
tools/news.py
tools/weather.py
tools/browser_bot.py
4

Smart home

TV, AC, lights, curtains via Raspberry Pi

Connect a Raspberry Pi to relay modules wired to your home devices. Friday sends MQTT messages, Pi executes GPIO signals to control the hardware.

Hardware needed:
Raspberry Pi 4 · Relay module board · IR blaster (for TV/AC) · Servo motors (curtains) · Jumper wires

Install:
pip install paho-mqtt pip install RPi.GPIO (on Pi)

Tools to write:
tools/smart_home.py
pi/listener.py (runs on the Pi)
